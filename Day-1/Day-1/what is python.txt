1. What is Procedural Programming Language.
   Is a programming language that solves problems using step-by-step instructions and functions/procedures.
                          
                                      Input → Process → Output

2.What is Object Oriented Programming Language.
   Is a programming language that uses classes and objects to organize data and functions together.
                             
                                      Data → variables/attributes
                                      Functions → methods/behaviors
           
        Main concepts of OOP.
            
           1.Class – Blueprint for objects
           2.Object – Real instance of a class
           3.Encapsulation – Hiding data for security
           4.Inheritance – One class acquires properties of another class
           5.Polymorphism – Same function behaves differently
           6.Abstraction – Showing important details and hiding unnecessary details.

3. What is Python ?
     Python is a high-level, easy-to-learn programming language used for web development, data science, artificial intelligence, automation,           and software development.

4.Why Python ?
     Python is used because it is simple, easy to learn, readable, and supports many applications like web development, AI, automation, and data science.
